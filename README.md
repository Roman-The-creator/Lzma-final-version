# LZMA Archiver (Python)

Проект представляет собой реализацию архиватора, использующего алгоритм сжатия LZMA.

## Требования
Для работы проекта требуется:

Python версии 3.7 или выше.

Стандартные библиотеки Python (дополнительная установка внешних пакетов не требуется).

## Структура проекта

lzma_compressor.py — ядро алгоритма. Содержит реализацию Range Encoder, Range Decoder и логику работы с литеральными контекстами.

archiver_lzma.py — модуль управления архивом. Отвечает за формирование заголовков, упаковку нескольких файлов в один контейнер и проверку целостности через CRC32.

main_lzma.py — консольный интерфейс (CLI) для взаимодействия с пользователем.

verify_archiver_lzma.py — скрипт для проведения автоматизированных интеграционных тестов.

tests_lzma.py — набор модульных тестов для проверки отдельных компонентов системы.

Запуск программы осуществляется через файл main_lzma.py.
